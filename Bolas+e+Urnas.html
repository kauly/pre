Introdução¶

Considere uma urna com t bolas, sendo g verdes e r vermelhas, com g + r = t. Quando uma bola retirada com reposio, significa que esta bola volta para a urna antes da prxima ser retirada. Quando uma bola retirada sem reposio, significa que esta bola no volta para a urna antes da prxima ser retirada.
Sejam os seguintes experimentos probabilsticos, em que n e k so parmetros dos experimentos.

  • Experimento 1a. Retira-se, uma por uma, um total de n bolas da urna, com reposio.
  • Experimento 1b. Retira-se, uma por uma, um total de n bolas da urna, sem reposio.
  • Experimento 2a. Retira-se, uma por uma, bolas da urna, com reposio. O experimento se encerra imediatamente aps a k-sima bola verde retirada.
  • Experimento 2b. Retira-se, uma por uma, bolas da urna, sem reposio. O experimento se encerra imediatamente aps a k-sima bola verde retirada

Defina as seguintes variveis aleatrias:

  • Experimentos 1a e 1b. K = nmero de bolas verdes retiradas no experimento.
  • Experimentos 2a e 2b. N = nmero total de bolas retiradas no experimento.
    Tais variveis aleatrias esto relacionadas com as seguintes distribuies discretas:
  • Experimentos 1a. Distribuio binomial.
  • Experimentos 1b. Distribuio hipergeomtrica.
  • Experimentos 2a. Distribuio binomial negativa (ou de Pascal).
  • Experimentos 2b. Distribuio hipergeomtrica negativa.

Dados:

n = numero de bolas retiradas da urna
t = nmero de bolas
g = bolas verdes
r = bolas vermelhas
t = g+r
k = possivel resultado da variavel aleatoria
K = nmero de bolas verdes retiradas no experimento.

$$p=\frac{g}{t} \quad q=\frac{r}{t} \qquad \binom{n}{k} = \frac{n!}{k!(n-k)!}$$

PMF 1a - $$p_K(k)=\binom{n}{k}p^kq^{n-k}$$

PMF 1b - $$p_K(k)=\frac{\binom{g}{k}}{\binom{t}{n}}\binom{r}{l}$$

def coef_binon(n,k):
  
    if k > n:
        return 0
    return math.factorial(n)/(math.factorial(k)*math.factorial(n-k))

def prob(g,t): 
    return float(g)/t

def pmf_1a(p,n,k):
    pmf = coef_binon(n,k)*(p**k)*(1-p)**(n-k)
    return pmf

def pmf_1b(g,r,t,n,k):
    l = n-k
  
    pmf = (coef_binon(g,k)/coef_binon(t,n))*coef_binon(r,l)
    return pmf

def media_1(n,p):
    return n*p

def var_1a(n,p,q):
    return n*p*q

def var_1b(n,p,q,t):
    return (n*p*q)*(t-n/t-1)

def exp_1a(t,g,n):
     p = prob(g,t)
     
     i = 0
     outcomes = []
     while i < n:
            a = random.uniform(0, 1)
            outcomes.append(a <= p)
            i += 1       
     return np.sum(outcomes)

def exp_1b(t,g,n):
     i = 0
     outcomes = []
     while i < n:
            a = random.uniform(0, 1)
            p = prob(g,t)
            outcomes.append(a <= p)
            t -= 1
            i += 1       
     return np.sum(outcomes)

Experimento 2 a/b¶

  • Experimento 2a. Retira-se, uma por uma, bolas da urna, com reposio. O experimento se encerra imediatamente aps a k-sima bola verde retirada.
  • Experimento 2b. Retira-se, uma por uma, bolas da urna, sem reposio. O experimento se encerra imediatamente aps a k-sima bola verde retirada

VA's:

  • Experimentos 2a e 2b. N = nmero total de bolas retiradas no experimento.

PMF's:

  • Experimentos 2a. Distribuio binomial negativa (ou de Pascal).
  • Experimentos 2b. Distribuio hipergeomtrica negativa.

Formulas:

$$p=\frac{g}{t} \quad q=\frac{r}{t} \qquad \binom{n}{k} = \frac{n!}{k!(n-k)!}$$

PMF 2a - $$p_K(k)=\binom{n-1}{k-1}p^kq^{n-k}$$

PMF 2b - $$p_K(k)=\frac{\binom{g}{k-1}}{\binom{t}{n-1}}\frac{g-k+1}{t-n+1}\binom{r}{l}$$

Media 2a - $$E[N]=\frac{k}{p}$$

Media 2b - $$E[N]=k\frac{t+1}{g+1}$$

Varincia 2a - $$var[N]=k\frac{q}{p^2}$$

Varincia 2b - $$var[N]=k\frac{(t-g)(t+1)(g+1-k)}{(g+1)^2(g+2)}$$

def pmf_2a(n,k):
    return coef_binon(n-1,k-1)*(p**k)*(1-p)**(n-k)

def pmf_2b(t,g,n,k):
    return (coef_binon(g,k-1)/coef_binon(t,n-1))*((g-k+1)/(t-n+1))*coef_binon(t-n,n-k)

def media_2a(p,k):
    return k/p

def media_2b(t,g,k):
    return k*(t+1/g+1)

def var_2a(p,k):
    q = 1-p
    return k*(q/p**2)

def var_2b(t,g,k):
    return k*(((t-g)*(t+1)*(g+1-k))/(((g+1)**2)*(g+2)))

def exp_2a(t,g,n):
     p = prob(g,t)
     i = 1
     while i < n+1:
            a = random.uniform(0, 1)
            if a <= p:
                return i
            i += 1       
     return 0

def exp_2b(t,g,n):
     i = 0
     outcomes = []
     while i < n:
            a = random.uniform(0, 1)
            p = prob(g,t)
            outcomes.append(a < p)
            t -= 1
            i += 1       
     return np.sum(outcomes)
